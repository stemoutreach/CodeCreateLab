
# 🎯 Level 1 – Challenge Checklist

Welcome to your first set of challenges in the **Code and Create Lab**! To unlock Level 2, complete **both setup and Python basics** by finishing these activities from **Lesson 1A** and **Lesson 1B**.

---

## ✅ Part A: Raspberry Pi Setup

Demonstrate that you can:

1. ✅ Connect your Raspberry Pi with monitor, keyboard, mouse, and power.
2. ✅ Boot up the Raspberry Pi and finish the welcome setup screen.
3. ✅ Connect to Wi-Fi.
4. ✅ Open the terminal and run this command:

```bash
sudo apt update && sudo apt full-upgrade
```

5. ✅ Find and open Thonny from the Raspberry Pi Menu.

---

## ✅ Part B: Python Basics

Write and run the following Python scripts in Thonny:

### 📄 Script 1: Hello & Math
```python
print("Hello from [Your Name]")
print(5 + 3)
```

### 📄 Script 2: Favorite Color
```python
color = input("What is your favorite color? ")
print("You chose", color)
```

### 📄 Script 3: Greeting and Age
```python
name = input("What is your name? ")
age = input("How old are you? ")
age = int(age)
print("Hi", name + "! In 10 years you’ll be", age + 10)
```
