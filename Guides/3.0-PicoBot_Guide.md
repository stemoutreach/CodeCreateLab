### 3. Adding Ultrasonic Sensor

**Objective:**
Add sensor reading capabilities.

**Ultrasonic Function Example (`robot_utils.py`):**

```python
from machine import Pin, time_pulse_us

trigger = Pin(14, Pin.OUT)
echo = Pin(15, Pin.IN)

def read_distance():
    trigger.low()
    time.sleep_us(2)
    trigger.high()
    time.sleep_us(10)
    trigger.low()
    duration = time_pulse_us(echo, 1)
    distance_cm = duration / 58.0
    return distance_cm
```

## Objective

## Prerequisites

* Completed wiring and testing of motors
* Working ultrasonic distance sensor

## Instructions

1. **Setup Project Structure:**
   Create two files:

* `robot_utils.py` for reusable functions
* `main.py` for running your robot

2. **Implement Movement Functions:**
   Complete functions for moving forward, backward, turning, and stopping.

3. **Add Sensor Functionality:**
   Complete your ultrasonic sensor reading function.

4. **Maze Navigation Algorithm:**
   Use sensor data to decide robot movements.
