### 4. Maze Navigation Program

**Objective:**
Develop a complete program to autonomously navigate through a maze.

**Starter Code (`main.py`) with ToDos:**

```python
import robot_utils
import time

SAFE_DISTANCE = 10  # centimeters

# Main maze navigation function
def navigate_maze():
    while True:
        # TODO: Use your ultrasonic sensor to read distance
        distance = robot_utils.read_distance()

        # TODO: Implement logic to turn left if too close to an obstacle
        if distance < SAFE_DISTANCE:
            robot_utils.stop_motors()
            robot_utils.turn_left()
            # TODO: Adjust sleep duration for correct turning
            time.sleep(0.4)
        else:
            robot_utils.move_forward()

        # TODO: Add additional sensor checks for maze completion
        time.sleep(0.1)

if __name__ == "__main__":

        navigate_maze()
    finally:
```
