# Code & Create Python Projects

Welcome to the **Code & Create Python Projects**! This collection of beginner-friendly coding challenges is designed to help 4th–10th grade students learn Python in a fun, hands-on way.

Each project builds on core skills like:
- Printing messages
- Using variables
- Writing functions
- Controlling flow with conditionals and loops
- Drawing with Turtle
- Working with strings and lists

## ✅ How to Use These Projects

1. Pick a project that looks interesting.
2. Open the `.md` file and read the challenge.
3. Use the `# TODO:` checkboxes to complete the task in your Python file.
4. Ask for help or check the Python reference project if you get stuck!

Happy coding! 💻🐍
