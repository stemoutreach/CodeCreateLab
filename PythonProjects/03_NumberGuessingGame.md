# Number Guessing Game

## Description
Let the user guess a secret number. Give hints and count tries.

## What To Do
- [ ] Generate a random number using import random.
- [ ] Use a while loop to let the user guess until correct.
- [ ] Tell the user if the guess is too high or low.
- [ ] Count the number of tries.
- [ ] Challenge: Add hints like 'warmer' or 'colder'.
