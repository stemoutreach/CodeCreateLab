# 3.0 PicoBot Lab: Maze Explorer 

## Objective

Program your PicoBot to navigate autonomously through a maze using the ultrasonic sensor.

## Prerequisites

* Completed wiring and testing of motors
* Working ultrasonic distance sensor

## Instructions

1. **Setup Project Structure:**
   Create two files:

* `robot_utils.py` for reusable functions
* `main.py` for running your robot

2. **Implement Movement Functions:**
   Complete functions for moving forward, backward, turning, and stopping.

3. **Add Sensor Functionality:**
   Complete your ultrasonic sensor reading function.

4. **Maze Navigation Algorithm:**
   Use sensor data to decide robot movements.

**Starter Program (`main.py`):**

```python
import robot_utils
import time

SAFE_DISTANCE = 10  # centimeters

# Main maze navigation function
def navigate_maze():
    while True:
        distance = robot_utils.read_distance()
        if distance < SAFE_DISTANCE:
            robot_utils.stop_motors()
            robot_utils.turn_left()
            time.sleep(0.4)  # Adjust for correct turning
        else:
            robot_utils.move_forward()

        # TODO: Add code to check for maze completion (e.g., line sensor)
        time.sleep(0.1)

if __name__ == "__main__":
    try:
        navigate_maze()
    finally:
        robot_utils.stop_motors()
```
