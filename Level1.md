# Level 1 – Pi Setup & Python Intro

## 🧠 Learning Objectives

* Identify and connect Raspberry Pi components (monitor, keyboard, mouse, power)
* Boot into Raspberry Pi OS
* Connect to Wi-Fi
* Open and use the Thonny Python IDE
* Write and run your first Python script

## 🧰 Materials Needed

* Raspberry Pi 3 or later
* Power supply
* Monitor, keyboard, and mouse
* MicroSD card with Raspberry Pi OS
* Internet connection (optional for updates)

## 📝 Instructions

1. **Assemble Hardware**: Connect the monitor, keyboard, mouse, and power to your Raspberry Pi.
2. **Boot the Pi**: Insert the SD card, power on, and wait for the OS to load.
3. **Connect to Wi-Fi**: Use the taskbar network icon to select and connect to a network.
4. **Launch Thonny**: Open the Thonny IDE from the application menu.
5. **Hello, World!**: Type `print("Hello, world!")` and run the script.

## 🧪 Mini Challenge

Create a Python script that:

* Prints your name
* Prints the result of 5 + 3
* Uses `input()` to ask the user their favorite color

## ✅ Checkpoint

To pass this level, demonstrate:

* A working Pi setup
* A Python script that prints, calculates, and asks for input
