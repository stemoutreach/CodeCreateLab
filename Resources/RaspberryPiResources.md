# 📚 Raspberry Pi References

A curated list of beginner-friendly resources for learning Raspberry Pi—covering books, websites, videos, and hands-on kits.

---

## 📘 Books

### Ages 8–12
- **_Adventures in Raspberry Pi_** by Carrie Anne Philbin  
  Fun, hands-on projects. Written by an educator from the Raspberry Pi Foundation.

- **_Raspberry Pi for Kids_** by Jason Briggs  
  Covers Scratch and Python with clear explanations and colorful illustrations.

### Ages 10–14
- **_Coding Projects in Python_** by DK  
  Python-based games and applications with visual, step-by-step instructions.

- **_Learn Robotics with Raspberry Pi_** by Matt Timmons-Brown  
  Ideal for students interested in robotics. Written by "The Raspberry Pi Guy" on YouTube.

---

## 💻 Web Resources

### Official & Kid-Friendly Sites
- **[Raspberry Pi Projects](https://projects.raspberrypi.org/en/)**  
  Free, step-by-step guides. Filter by age, skill level, and programming language.

- **[Code Club Projects](https://codeclubprojects.org/en-GB/)**  
  Great for ages 9–13. Offers Scratch, Python, and physical computing projects.

- **[TinkerGen Learn](https://www.tinkergen.com/learn/)**  
  Beginner hardware projects using MakeCode and Python.

---

## 📺 YouTube Channels

- **[Raspberry Pi Foundation](https://www.youtube.com/user/RaspberryPiFoundation)**  
  Official tutorials and inspirational stories.

- **[The Raspberry Pi Guy](https://www.youtube.com/user/theRaspberryPiGuy)**  
  Easy-to-follow projects and hardware demos.

- **[KidsCanCode](https://www.youtube.com/c/kidscancode)**  
  Python and game development tutorials suitable for middle schoolers and up.

---

## 🧰 Raspberry Pi Kits

- **[Piper Computer Kit](https://www.playpiper.com/)**  
  Build your own computer and code using Minecraft. Aimed at ages 8–13.

- **Kano Computer Kit** (discontinued but available online)  
  Raspberry Pi-based kit with a child-friendly OS, block coding, and tutorials.

---

## 🏫 Online Learning Platforms

- **[Tynker](https://www.tynker.com/)**  
  Includes Raspberry Pi integrations with block coding and Python.


---

For structured learning experiences, many of these resources can be adapted for home use, coding clubs, or classrooms.
