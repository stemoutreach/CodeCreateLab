## 🧩 Functions

### What is a Function?
A function is a reusable block of code. You define it once and use (call) it many times.

```python
def greet(name):
    print("Hello", name)

greet("Ada")
greet("Alan")
```

### Function with Return Value

```python
def add(x, y):
    return x + y

result = add(5, 3)
print("5 + 3 =", result)
```
