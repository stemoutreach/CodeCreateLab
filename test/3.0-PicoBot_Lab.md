# Lab 3.0 – PicoBot Maze Explorer

## Overview
Program your PicoBot to autonomously find a path through a cardboard maze using its front ultrasonic sensor.

## Learning objectives
- Combine motor control APIs and distance sensing
- Implement a simple wall‑following or “follow‑the‑right‑hand” algorithm
- Create reusable helper functions (`drive_forward`, `turn_left`, etc.)
- Tune speed constants for reliable movement

## Prerequisites
- Lab 2 completed
- Wiring verified per **Guides/PicoBot_Guide.md**

## Maze rules
- Walls are at least 15 cm tall.
- A **finish line** is marked with black tape across the floor.

## Starter code skeleton

```python
from picobot import motors, ultrasonic   # provided library
import time

SAFE_DISTANCE = 10    # cm
BASE_SPEED    = 60    # 0‑100

def drive_forward(speed=BASE_SPEED):
    motors.set(speed, speed)

def turn_right():
    motors.set(BASE_SPEED, -BASE_SPEED)
    time.sleep(0.4)

def turn_left():
    motors.set(-BASE_SPEED, BASE_SPEED)
    time.sleep(0.4)

def stop():
    motors.stop()

def main():
    while True:
        dist = ultrasonic.read_cm()

        if dist < SAFE_DISTANCE:
            stop()
            turn_left()            # simple left‑hand rule
        else:
            drive_forward()

        # TODO: check line sensor for black finish‑line tape
        time.sleep(0.05)

if __name__ == "__main__":
    try:
        main()
    finally:
        stop()
```
