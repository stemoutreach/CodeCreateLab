# Level 2 – Python Basics

## 🧠 Learning Objectives

* Use Python variables and data types
* Understand and use loops (for, while)
* Use conditionals (if/else)
* Write and call functions

## 🧰 Materials Needed

* Raspberry Pi
* Thonny Python IDE

## 📝 Instructions

1. **Variables**: Practice assigning values to variables and printing them.
2. **Loops**: Write a `for` loop to print numbers 1–10.
3. **Conditionals**: Write an `if` statement to check a number and print if it's even or odd.
4. **Functions**: Define a function that adds two numbers and returns the result.

## 🐢 Optional Extension

Use Python’s `turtle` module to:

* Draw a square using a loop
* Experiment with colors and shapes

## 🧪 Mini Challenge

Write a script that:

* Asks for a user’s age
* Uses conditionals to print if the user is a child, teen, or adult
* Draws a shape using `turtle` if available

## ✅ Checkpoint

To pass this level, demonstrate:

* Working script with at least one loop, one conditional, and one function
* Optional: turtle drawing
